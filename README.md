##Blast (1.1.1)

**Docs**  
[julian.com/research/blast](http://julian.com/research/blast)

**Quickstart**

```sh
npm install blast-text
```

```sh
bower install blast-text
```

**Frameworks**  
Both jQuery and Zepto are fully supported.

**Browsers**  
All major browsers are supported. Back to IE6.
